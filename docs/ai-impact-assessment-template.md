# AI Impact Assessment Template

Use this template for every AI/ML system before deployment.

## System Description

- Name, version, intended purpose
- Users affected (internal / customers / third parties)
- Decision type (automated / recommender / advisory)

## Data

- Training-data sources and lawful basis
- Sensitive-attribute handling
- Retention schedule

## Risk Analysis

- Potential harms (individual, group, societal)
- Severity × likelihood rating
- Mitigations applied

## Sign-off

- AI Governance Committee decision: Approved / Conditional / Rejected
- Conditions / follow-up actions
- Next review date
